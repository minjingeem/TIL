<h5>Basic Level</h5>

1. Write a program that takes two integers A and B as input and outputs <b style="color:blue;">A+B</b>

```java
import java.util.*;

public class Main {
  public static void main(String[] args){
    Scanner scan = new Scanner(System.in);
    int a, b;
    a = scan.nextInt();
    b = scan.nextInt();
    System.out.println(a + b);
  }
}

```
